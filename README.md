# World Cup scoreboard

This is a free static site: no server, database, account, or paid service is required.

To share it publicly with GitHub Pages:

1. Create a GitHub repository and upload this folder's files.
2. In the repository, open **Settings → Pages**.
3. Choose **Deploy from a branch**, then select `main` and `/ (root)`.
4. GitHub will provide a public URL to share.

`data.js` contains the scores and point-by-point audit data. `app.js`, `styles.css`, and `index.html` are the interface.
